## 개요
- 프로젝트 이름: SearchUnion
- 장르 : 공포, 시뮬레이션
- 개발 엔진 및 언어: Unity v.2022.03.24f1 & C#

<br/>
  
## 게임 플레이 방식
캐릭터 이동 방법    
- 이동 : w, a, s, d
- 제자리 뛰기 : space
- 달리기 : shift
- 웅크리기 : ctrl
- 손전등 사용 : F
- 인벤토리 : Tab
- 상호작용 : E
